#### Установка зависимостей

```bash
yarn
```

или

```bash
npm install
```



#### Запуск в режиме разработки

```bash
yarn dev
```

или

```bash
npm run dev
```

---

#### Сборка проекта

```bash
yarn build
```

или

```bash
npm run build
```

#### Production

```bash
yarn start
```

или

```bash
npm run start
```
